# A Response to NYC's Preliminary Citywide Racial Equity Plan

**🔗 View the live page:** https://dont-shoot-the-data.github.io/nyc-racial-equity-response/

A data response to NYC Mayor Mamdani's Preliminary Citywide Racial Equity 
Plan, released April 6, 2026. Every chart on this page is sourced directly 
from primary sources you can verify yourself:

- NYPD Annual Enforcement Reports (2008–2024)
- NYC Open Data — NYPD Arrests Historic
- Columbia University Poverty Tracker
- The Mayor's own True Cost of Living measure
- Vital City NYC analysis of NYPD/DCJS conviction data

## What's in this repo

- `index.html` — the full page, single file, no dependencies beyond Chart.js CDN

## Sources

All sources are linked directly from the live page. No data is hidden, 
no calculations are obscured, and every claim reduces to a primary source.
